# nelson.social/bots

Posts info to the following accounts:

- https://nelson.social/@WhitewaterBot
- https://nelson.social/@AvalancheBot

# Prerequisites

* Node.JS 16+

# Install

```bash
npm install
```

# Run
```bash
MASTODON_TOKEN=<token> MASTODON_API=https://nelson.social/api/ npm run post-status (whitewater|avalanche)
```
